---

## 📖 Overview

This repository contains my **Footprinting & Reconnaissance** laboratory project completed as part of my **Cyber Security coursework**.

The purpose of this project was to understand how cybersecurity professionals gather publicly available information during the reconnaissance phase of a security assessment. The lab demonstrates ethical information gathering techniques using Open Source Intelligence (OSINT), DNS analysis, WHOIS lookups, website reconnaissance, and network enumeration.

All activities were performed in a controlled educational environment and only against authorized targets.

---

## 🎯 Project Objectives

* Perform Search Engine Footprinting
* Conduct Website Footprinting
* Perform Email Footprinting
* Execute WHOIS Enumeration
* Analyze DNS Records
* Conduct Network Footprinting
* Collect Open Source Intelligence (OSINT)
* Perform Reconnaissance Using Industry Tools
* Document Findings Professionally
